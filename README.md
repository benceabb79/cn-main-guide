# cn-main-guide

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容涵盖技术参考、工具说明、学习笔记等，不针对任何特定域名或网站。

## 项目简介

本仓库主要收集和整理一些独立的 HTML 文件，旨在提供一种简单的页面归档和发布方式。每个页面均为自包含的 HTML 文档，可直接在浏览器中打开查看。

## 目录说明

```
cn-main-guide/
├── pages/           # 存放各独立 HTML 页面
├── assets/          # 页面引用的资源文件（图片、样式、脚本等）
├── README.md        # 本文件
└── ...
```

- `pages/`：每个子目录或文件对应一个独立页面，命名尽量反映内容主题。
- `assets/`：按页面或类型组织资源，避免重复引用。

## 页面归档说明

- 所有页面均为静态 HTML，无需后端支持。
- 页面内容可能涉及技术文档、工具使用、学习笔记等，来源多样。
- 页面之间无强关联，可独立查看。
- 部分页面可能包含外部资源（如 CDN 引用的库），请确保网络连接正常。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时会通过提交记录体现。
- 欢迎通过 Issue 提出建议或报告问题，但请注意不要包含具体域名或营销内容。
- 不接受直接针对本仓库的推广或广告类贡献。
- 对于页面内容的准确性，维护者会尽量核实，但不做严格保证。

## 使用方式

1. 克隆或下载本仓库。
2. 在 `pages/` 目录下找到感兴趣的 HTML 文件。
3. 直接在浏览器中打开即可查看。
